About Base 64

What is Base64?

Base 64 is used to encode and decode data. It can convert binary to ascii and vice versa. Base64 includes characters: from A to Z, a - z, 0 - 9, + and /. Total is 64 characters. Character from A to Z represent values from 0 to 25, a-z represent value from 26 to 51, 0-9 represent values from 52 to 61, + represent value 62, - represent value 63. In base64, a character use 6 bits
