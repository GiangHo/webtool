About all tools online for developers

This site is created to help person can code easier.

I often feel uncomfortable when debug, test, check with nonstandard data. So I develop this tool to work easier for me and for someone. This tool is build and develop gradually. At present, it has some tools about converter, encode/decode, formatter, generator, string tools …

About formatter

These tools are developed to help you have a data beautiful visualization. Your data is in html, json, xml, sql format.

Example:

You have json:

{"name":{"firstname":"Emma","lastname":"Watson"}}

Json after the tool handles:

{
   "name":{
      "firstname":"Emma",
      "lastname":"Watson"
   }
}

About Converter

Sometimes, you know how to create, change, convert somethings. But it’s not necessary, you just want to result for help your work. For example, you want to convert binary to decimal. You don’t care how or why it can convert. You just care result. You can use this tool to convert.

Example:

You have binary value: 1010

You will have decimal value: 10

About Encode/Decode

Sometimes during work, you need encode or decode. You can encode/decode url, base64, … with this tool.

Example:

Your string need to encode base64: th1s 1s p3ssw0rd

After encode, dGgxcyAxcyBwM3NzdzByZA==

About Generator

Security is important. If you need using key string when send data, you can use MD5 hash or SHA hash.

Example:

Your string: 18122018hashthiskey

MD5 hash: 3e785a69630ac0a9d9906e4b257ed8d3

About string tools

If you want count, reverse string or remove extra spaces, remove line breaks, remove special character, you can do it with this site.

Example:

Your string: Monday is today

After reverse: Today is Monday.
