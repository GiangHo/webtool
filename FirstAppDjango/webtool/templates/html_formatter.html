About HTML

What is HTML?

HTML stands for Hyper Text Markup Language. It’s used for creating web pages and web applications. HTML includes elements. HTML elements are tags with separated meaning.

Example:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>
</html>
